# 一下（Project001 · Once）

一个用来**感受时间**的秒表。

---

## 项目简介

“一下”是一种极简的计时器，也是一种时间训练工具。它不追求精度，而追求**感觉**。每一次开始与停止的动作，都是一次和时间的对话。你可以在开始前设定一个目标时长，比如「我猜刷牙需要 2 分 30 秒」。当你按下停止键，系统会告诉你这次与目标的差距——越接近目标，就越接近那个甜区（sweet spot）。

若你不设定目标，它依然会记录当前时长，并显示与上一次计时的差值。久而久之，你会自然地形成一种对时间的直觉校准，就像肌肉记忆一样。

这个项目的初衷并不是效率，而是**让时间重新变得可感**。对于注意力容易漂移或时间感模糊的人（例如 ADHD 倾向者），它是一种温和的练习——通过一连串小的强化反馈，让大脑重新建立“估计 → 执行 → 反馈”的通路。

---

## 功能与交互

* **开始 / 暂停 / 重置**：核心功能，所有反馈都源于一次按下。
* **目标时长设定（可选）**：输入预期时间，用于 sweet spot 评分。
* **结果显示**：

  * 本次计时
  * 上一次计时与差值
  * 若设定目标，显示偏差与评分
* **sweet spot 算法**：
  评分基于误差比例，例如误差 ≤5% 计为 100 分，随误差增大平滑衰减。
* **视觉反馈**：
  接近目标时呈现轻柔的亮色动画，偏离时则安静地变冷。

---

## 设计理念

“一下”不是为了帮你节省时间，而是帮你**重新体会时间的密度**。
按下按钮的那一刻，是一个决策点；松开的那一刻，是一次反思。
它既可以用于微小的生活片段（泡茶、刷牙、洗脸），也可以用于抽象的思考练习。

在设计上，“一下”追求可复现性与轻盈感：任何人都能在 60 分钟内搭建出网页版原型，
不需要后端，不需要账户，只需打开页面、输入目标、按下按钮。

## 当前阶段

* **阶段一：网页原型**
  使用 HTML + JS 实现基本逻辑与评分动画。
* **阶段二：实体版本（可选）**
  以 ESP32 或 RP2040 为核心，使用实体按键、屏幕与光/振动反馈构建可握持设备。
